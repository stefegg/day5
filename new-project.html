Starting a new project

1. Create a Project folder

You can use the command $ mkdir (folder-name) or you can do this through the interface

2. Initialize Git Repository in project

Navigate to the project folder inside the terminal and run the following command:

$ git init

You should see the following output:

Initialized empty Git repository in (pathname)

3. Open a browser and go to Github and create a new repository.

Once you create a new Repository, you will copy and paste the following commands into the terminal. These commands set the remote to point to your project folder ...


$ git remote add origin https://github.com/stefegg/(folder)

as well as starting a new branch for you to work on. This branch should be called master.


$ git push -u origin master

you can combine these by using && with spaces in between

You're all set!

Now read about adding files to your project and pushing your code to Github!
